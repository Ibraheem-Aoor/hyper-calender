Required Software

This application is build with Laravel 8. The server must meet the following requirements to run this application. See the details documentation for Laravel

  • PHP >= 7.3
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

 

Creating Database

Create a database with any name you want. If you are using phpMyAdmin go to browser open http://localhost/phpmyadmin and click on "New" from left the panel.

Setting .env file

Open .env file located on top of the directory. Enter the database name, database username and database password as shosn below.


If you want to import the database with dummy data by uploading .sql file ignore step 4 & 5 and skip to step 6.

Migrate Tables

After creating database and setting .env file migrate tables to database. You can simply achieve this by running this command on console:

     php artisan migrate

Insert Data

All search keys and options are dynamically taken from database in "panjika". The filtering option will be empty if there is no data in database.

Import Database

If you performed step 4 & 5 ignore this step.

The database dump file named panjika.sql is in sql folder. Just import the file to associated database.

File Structure

The default structure of this project is same as Laravel's default directory structure. Below are some important file structures that have been created for this project.

  1. Create Custom Command: /app/Console/Commands
  2. Controller Files: /app/Http/Controller
  3. Model Files: /app/Model
  4. Providers Files: /app/Providers
  5. Repositories: /app/Repositories
  6. View Files: /resources/views
